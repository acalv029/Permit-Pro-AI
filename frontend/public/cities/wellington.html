⚠️ Common Rejection Reasons — Wellington

These are the most common reasons permit packages get rejected in Wellington. Our AI flags all of these automatically.

  • ⚠️ Application must be signed by BOTH property owner AND contractor — missing either signature = rejection
  • ⚠️ Insurance cert holder must read EXACTLY: 'Village of Wellington / 12300 Forest Hill Blvd / Wellington, FL 33414' — any variation rejected
  • ⚠️ Contractor Registration Application MUST BE NOTARIZED — unsigned/unnotarized = rejected at registration
  • ⚠️ Registration requires $5,000 surety bond payable to the municipality per Ordinance 1983-19
  • ⚠️ NOC must be recorded AND posted before first inspection — NO subsequent inspections without certified copy — this is strictly enforced
  • ⚠️ HVAC NOC threshold is $15,000 (NOT the standard $5,000) — contractors often file NOC unnecessarily for small AC changeouts
  • ⚠️ Aggregate value = TOTAL COST of ALL improvements, not just the individual permit work — if total project across all permits ≥ $5,000, fee simple titleholder/bonding co/architect/mortgage lender info required
  • ⚠️ ALL plans/docs must be PDF format — other formats cause issues in ProjectDox ePlans system
